# ticket-price-monitoring-tg-bot

A smart Telegram bot that monitors flight ticket prices, analyzes historical data, and alerts you about the best moments to buy tickets at optimal prices.

```bash
python -m venv venv
pip install "apache-airflow==3.1.3" --constraint "https://raw.githubusercontent.com/apache/airflow/constraints-3.1.3/constraints-3.12.txt"
pip install apache-airflow-providers-postgres
pip install -r requirements.txt
```
